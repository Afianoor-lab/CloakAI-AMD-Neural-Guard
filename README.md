#  Cloak AI: Neural-Powered PII Masking & Security Intelligence

**Cloak AI** is a high-performance cybersecurity dashboard designed to protect sensitive data (PII/PCI) using the power of **AMD Instinct™ MI300X** and **Groq LPU** inference. It provides real-time "Neural Scrubbing" to ensure compliance with GDPR and HIPAA standards.

##  Key Features
- **Advanced Masking Engine:** High-speed neural scrubbing for names, emails, API keys, and financial IDs.
- **Cyber-Security AI Agent:** An intelligent assistant powered by Llama-3.3 (via Groq) to analyze security risks and provide expert advice.
- **Real-Time Risk Meter:** Dynamic threat level detection based on file analysis.
- **AMD Neural Engine Support:** Optimized for low-latency processing using AMD Developer Cloud infrastructure.

##  Technology Stack
- **Hardware:** AMD Instinct™ MI300X (via AMD Developer Cloud)
- **Inference Engine:** Groq LPU (Llama-3.3-70b-versatile)
- **Framework:** Streamlit, Python
- **API Integration:** Groq Cloud API, AMD ROCm Ready
